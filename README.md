## ðŸ“¦ How to Push to GitHub & Deploy
To get your site live on `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`:

1. **Create a Repository**: Go to GitHub and create a new repository (e.g., `portfolio`).
2. **Initialize & Push**: Open your terminal in this project's root folder and run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Aditi Portfolio"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```
3. **Enable GitHub Pages**:
   - In your GitHub repository settings, go to **Pages** (under the "Code and automation" section).
   - Under **Build and deployment > Source**, ensure **Deploy from a branch** is selected.
   - Select the `main` branch and `/ (root)` folder, then click **Save**.
   - Your site will be live in about 1-2 minutes!
